# Our Little Planet
<img width="722" height="402" alt="Screenshot 2026-04-01 at 9 28 06 PM" src="https://github.com/user-attachments/assets/1361e0af-f1c4-4c3b-b8d9-ae108eaac41b" />

Our Little Planet is a 3D open world game that takes place in a rainforest. Players freely experience the world, discovering what nature has to offer. As players explore the rainforest, they realize that life isn’t as peaceful as they think. Acts of deforestation, overfishing, and fertilizer runoff have been destroying the ecosystem. By taking pictures, players can convince humans to unite against the destruction of the rainforest. They can re-plant trees, pick up litter, and rebuild habitats. In the end, players collect an assortment of photos, showcasing the change in the environment and the beauty of a balanced world.

This project was created as a submission to the Games for Change 2026 Student Challenge, in the All in for Nature: Reefs & Rainforests theme category. We wanted to make something ambitious and more long term than our other games so thats why we created this cozy open world game! Mainly, we created this just for fun and for others to enjoy :)

This project was made using Unity game engine, Visual Studio Code IDE, Blender for 3D modelling, MediBang Paint for 2D assets, and GitHub for collaboration.

I mainly struggled with learning new skills such as 3D modelling, 3D animations, shaders and shadergraphs, postprocessing effects. Basically all the 3D assets and polishing skills that I haven't used before! So this game helped me a lot in developing and learning these skills.
